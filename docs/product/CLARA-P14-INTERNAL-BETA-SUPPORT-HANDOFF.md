# CLARA P14 Internal Beta Support Feedback Handoff

## Support Scope

Support/feedback remains manual/local/repo-safe unless separately approved.
Known issues workflow is internal beta only. no external support tool
integration is activated.

## Privacy Rule

Secrets/tokens/cookies/auth headers/raw provider payload/raw webhook payload/raw
HTML/raw DOM/raw prompts/payment data must not be included in handoff, feedback,
logs, docs, or runbooks. Feedback should minimize customer-sensitive data.

Do not send email, Slack, Discord, webhook, push notification, or support tool
notification from this P14 handoff.
