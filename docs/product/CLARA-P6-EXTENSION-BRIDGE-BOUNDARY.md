# CLARA P6 Extension Bridge Boundary

The extension bridge is not an official provider replacement.

## Allowed

- User-assisted snapshot only.
- Active visible chat only.
- Bounded plain-text message snapshots.
- CLARA-authenticated snapshot upload.
- ChatGPT Companion preview/copy/open/manual only.

## Blocked

- No cookies.
- No session tokens.
- No provider tokens.
- No raw DOM.
- No raw HTML.
- No background crawling.
- No inbox crawling.
- No auto-send.
- no auto-send.
- No auto-submit to ChatGPT.
- No provider credential capture.

## Production Meaning

Extension bridge data can help operators during transition work, but it does
not satisfy production provider integration requirements. Production provider
work must follow the Official Channel Policy and Provider Readiness Matrix.
