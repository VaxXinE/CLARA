# CLARA P12 Known Limitations

CLARA is not GA yet and is not production deployed yet.

## Readiness-Only Features

- Billing, plan, entitlement, quota, and usage metering surfaces.
- Enterprise compliance, audit retention, backup/restore, incident response,
  evidence, and admin security surfaces.
- Queue/job/retry/dead-letter and alert readiness.
- Performance/load-test/capacity readiness.
- Provider health and scheduler readiness surfaces.

## Demo / Simulated / Review-Only Features

- Local demo auth and seeded demo data.
- Simulated provider paths where explicitly configured.
- Review-only AI surfaces.
- Human-approved reply and CRM proposal flows.
- Operator-assisted extension bridge and ChatGPT Companion preview/copy/manual
  flows.

## Explicitly Not Ready

- Public self-serve launch.
- Production billing or payment provider use.
- Real autonomous AI action.
- Unmanaged real provider sends.
- Evidence export, raw data export, or production automation execution.
