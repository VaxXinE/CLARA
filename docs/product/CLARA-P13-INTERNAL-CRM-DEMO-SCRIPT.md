# CLARA P13 Internal CRM Demo Script

## Demo Goal

Show that CLARA is usable for internal CRM workflow after P13 without claiming
production deployment, public GA launch, billing activation, real AI, or real
provider sends.

## Script

1. Start local API and dashboard.
2. Open dashboard as agent.
3. Show customer list.
4. Create a customer named `Demo CRM Customer`.
5. Update the customer status to `active`.
6. Add a short safe note.
7. Show the activity timeline entry.
8. Change lifecycle status to `follow_up`.
9. Assign owner to the demo agent.
10. Create a follow-up task.
11. Link an existing conversation to the customer.
12. Open the conversation and point out the linked customer.
13. Open the internal CRM dashboard analytics panel.
14. Switch to viewer and confirm read-only behavior.
15. State that billing/payment is deferred and public SaaS launch is deferred.

## Demo Safety

Do not paste secrets, real customer data, raw provider payloads, raw webhook
payloads, tokens, cookies, auth headers, raw prompts, or payment data into the
demo.
