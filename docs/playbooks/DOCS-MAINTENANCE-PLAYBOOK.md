# Documentation Maintenance Playbook

## When To Use

Use this when updating official Athena documentation.

## Steps

1. Identify affected document.
2. Read related standards and templates.
3. Update content.
4. Update navigation links.
5. Update version or changelog if needed.
6. Check security-sensitive content.
7. Request review.
8. Merge after approval.

## Security Check

Before merge, confirm:

- No secrets.
- No customer data.
- No private tokens.
- No sensitive screenshots.
