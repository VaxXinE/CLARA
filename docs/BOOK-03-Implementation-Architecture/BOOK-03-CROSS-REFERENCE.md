# By Review Type

## Architecture Review

Check:

- Boundaries.
- Ownership.
- Dependency direction.
- Data ownership.
- API contracts.
- Operational ownership.

## Security Review

Check:

- Authentication.
- Authorization.
- Tenant isolation.
- Secrets.
- Encryption.
- Audit logs.
- Threat model.
- Abuse cases.

## Quality Review

Check:

- Unit tests.
- Integration tests.
- Contract tests.
- E2E tests.
- Security tests.
- AI evaluation.
- Release gates.

## Operations Review

Check:

- Service owner.
- SLO/SLI.
- Dashboard.
- Alert.
- Runbook.
- Rollback.
- Incident path.
