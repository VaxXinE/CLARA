# Documentation Release Runbook

## Purpose

Safely release documentation updates.

## Steps

1. Confirm docs build/render.
2. Confirm links are valid.
3. Confirm navigation is updated.
4. Confirm no secrets.
5. Confirm related README/index files are updated.
6. Open PR.
7. Request review.
8. Merge.
9. Tag release if needed.
